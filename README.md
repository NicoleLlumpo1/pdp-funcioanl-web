
# PDP Funcional Web 

Este proyecto corresponde a la materia **Programación Funcional** y consiste en una aplicación web desarrollada con **HTML, CSS y JavaScript**.  
El objetivo es mostrar conceptos de programación funcional aplicados en un entorno web, con ejemplos prácticos y visuales.

---

##  Características
- Interfaz web simple y clara.
- Ejemplos de funciones y estructuras funcionales.
- Navegación intuitiva entre secciones.
- Código organizado y comentado para fines educativos.

---

##  Estructura del proyecto
- `index.html` → Página principal.
- `style.css` → Estilos del sitio.
- `script.js` → Lógica y ejemplos funcionales.
- `assets/` → Recursos adicionales (imágenes, íconos, etc.).
